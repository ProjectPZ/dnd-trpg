# ⚔️ DnD 5e 跑团 Skill

OpenClaw bot 的 DnD 5e 跑团插件。Bot 担任地下城主（DM），玩家在群聊中进行冒险。

## 快速开始

1. 群里说 `开始跑团` 或 `DnD`
2. 说 `创建角色` 建立你的冒险者
3. 跟随 DM 的叙述，描述你的行动
4. 说 `掷骰 1d20+5` 进行检定
5. 说 `结束跑团` 暂停游戏

## 游戏指令

| 指令 | 作用 |
|------|------|
| `开始跑团` / `DnD` | 进入游戏 |
| `创建角色` / `建卡` | 创建新角色 |
| `掷骰 XdY+Z` / `roll` | 掷骰子 |
| `角色卡` / `pc` | 查看角色信息 |
| `背包` / `inventory` | 查看物品 |
| `短休` / `长休` | 休息恢复 |
| `队伍` / `party` | 查看全队 |
| `结束跑团` / `收团` | 退出游戏 |

## 添加模组

在 `campaigns/` 目录下创建 `.txt` 文件，按模板格式编写即可。

## 文件结构

```
dnd-trpg/
├── SKILL.md              # 技能定义和完整规则
├── README.md
├── campaigns/            # 战役模组
│   └── starter.txt       # 入门冒险：破碗亭之谜
├── characters/           # 角色存档（自动生成）
└── scripts/
    ├── dice.py           # 骰子引擎
    ├── character.py      # 角色管理
    └── combat.py         # 战斗追踪
```
